# زنگ هشدار
زنگ هشدار، یک برنامه نوشته شده در زبان سی شارپ است که برای زنگ‌های هشدار طراحی شده است.

## ویژگی‌ها:
* ایجاد، تغییر وضعیت و حذف هشدارها
* انتخاب صدای زنگ هشدار سفارشی
* رابط کاربری ساده و...
